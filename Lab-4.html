Problem 1¶

1)¶

We are analyzing data from a satellite in a low Earth, 90 minute orbit that records the number of gamma rays it detects in a 100 ms interval. For each measurement, the metadata (see table above) gives us the GPS time, the position of the sun relative to the orbit, and the position of the satellite relative to the Earth. Since the data set consists of over 25 million data points, it would be impractical to try to discern anything about the cosmic ray background by looking at the entire set as a whole. However, we can use the fact that the satellite's orbit is periodic to partition the data set by orbital period. (The plot below of gamma ray counts vs. time demonstrates the periodic nature of the data.)

2)¶

Since the satellite is in a 90 minute orbit, we can analyze the background by averaging over all of the orbits. Below we've plotted how the average gamma-ray counts changes as the satellite progresses through its orbit. We can see that the number of counts drops off exponentially as a function of time. We've also calculated the standard deviation of the number of counts at each point in the orbit by partitioning the entire data set into 54,000 individual sets of about 480 elements since the satellite takes 54,000 measurements in each orbit and there are about 480 orbits total. Doing so, we found that the standard deviation also drops off exponentiall as a function of time.

3)¶

We were unable to perfectly model the background in time (see plots above). However, we suspect that the PDF of the background as a function of time would be a Gaussian function $N(x,t;\mu(t),\sigma^2(t))$ where $\mu(t) = A_\mu e^{-b_\mu t} + C_\mu$ and $\sigma^2(t) = \left[A_\sigma e^{-b_\sigma t} + C_\sigma\right]^2$. The reason we believe it is Gaussian is because of the plots below.

mod = np.remainder
t_0 = data[0,0]
def plot_random_time():
    t = np.round(time[np.random.randint(0,len(time))],decimals=1)
    gps = t_0 + t
    secs = int(mod(round(t * 10),54000))
    counts = part[:,secs]
    mu = counts.mean()
    sigma = counts.std()
    bins = int(max(counts) - min(counts))
    x = np.linspace(mu - 5*sigma, mu + 5*sigma,10000)
    gauss = stats.norm(loc=mu,scale=sigma)
    
    fig,ax = plt.subplots(1,1,figsize=(12,7))
    ax.hist(counts,bins=bins,density=True,label='Background PDF')
    ax.plot(x,gauss.pdf(x),label='N($\mu$,$\sigma^2$)')
    ax.set_ylabel('Probability',fontsize=16)
    ax.set_xlabel('Counts',fontsize=16)
    ax.set_title(f'GPS time = {gps} sec, $\mu$ = {np.round(mu,decimals=2)}, $\sigma$ = {np.round(sigma,decimals=2)}',fontsize=20)
    ax.legend(fontsize=22)
    plt.show()
    
for i in range(5):
    plot_random_time()

Above we have plotted the probability distribution of the number of counts at random times based on the phase of the satellite's orbit as well as a Gaussian PDF with the same $\mu$ and $\sigma$ as the background PDF. These graphs demonstrate that the background distribution is Gaussian with time-dependent parameters. The for of the time dependence of $\mu(t)$ and $\sigma^2$ follows from the results of 2).
